# Module 1: Practical Lab

## Lab Exercise: Creating Your First ROS 2 Package

### Objective
Create a ROS 2 package that implements a simple publisher-subscriber communication pattern.

### Prerequisites
- ROS 2 Humble Hawksbill installed
- Basic Python knowledge

### Steps
1. Create a new ROS 2 workspace
2. Create a package named `humanoid_control`
3. Implement a publisher node that publishes joint position commands
4. Implement a subscriber node that listens to sensor feedback
5. Test the communication between nodes

### Solution
We'll provide a complete example implementation that you can build upon for more complex humanoid robot applications.
