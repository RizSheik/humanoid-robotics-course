# Module 1 Assignment: ROS 2 Control Architecture

## Task

Design and implement a ROS 2 architecture for a humanoid robot's walking controller. Your solution should include:

1. A node for processing sensor data (IMU, joint encoders)
2. A node for computing walking gait commands
3. A node for sending commands to the robot's joint controllers
4. A visualization node to display the robot state

## Requirements

- Use appropriate message types for all communication
- Implement both service and action-based communication where appropriate
- Include error handling for communication failures
- Document the architecture with diagrams

## Submission

Submit your package with source code, launch files, and a documentation file explaining your design decisions.

## Evaluation Criteria

- Correct use of ROS 2 communication patterns
- Proper error handling and recovery
- Code quality and documentation
- Clear architectural decisions
