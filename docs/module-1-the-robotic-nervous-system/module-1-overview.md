# Module 1: The Robotic Nervous System

## Overview

This module covers the foundational elements of robot control and communication. We'll explore the middleware that enables communication between different components of a robotic system, focusing on the Robot Operating System 2 (ROS 2).

## Learning Objectives

By the end of this module, you will:
- Understand the architecture of ROS 2
- Master topics, services, and actions for robot communication
- Implement Python-based ROS 2 agents
- Create URDF models for humanoid robots
